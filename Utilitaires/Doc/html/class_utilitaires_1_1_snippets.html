DataSet Utilitaires.Snippets.UpdateInfoHits ( DataSet  dataSet,
int  info_id,
int  user_id 
)

Traitement de la table [0] cmsINFO puis identifie la ligne qui correspond à info_id et la mettre à jour.
Traitement de la table [1] : cmsCONTENT_HIT puis insertion de cette nouvelle ligne dans la table.

Côté console avant la fonction:
Dans console construire un dataset contenant 2 tables : cmsINFO et cmsCONTENT_HIT.
Envoyer le dataset + user_id + info_id au composant.
Apres la fonction:
Récupération du dataset et mise en cache.
A la fin de la session, mise à jour de la base données avec les infos contenus dans le dataset.
