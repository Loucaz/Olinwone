Utilitaires.DataServices Class Reference

Detailed Description

Classe pour travailler avec des requêtes SQL ainsi que ses paramètres destiner un interroger une base SQL serveur.
Exemple d’utilisation:
Constructeur: configure le SqlConnection et la SqlCommand
AddParameter (facultatif): ajouter des paramètres a la requête
GetStructures (facultatif): permet de faire des verifications
Execute (facultatif): execute la requette SQL
Dispose: Ferme le SqlConnection et la SqlCommand
Note
Il est conseiller de se renseigner sur System.Data.SqlClient avec notament SqlConnection et SqlCommand
